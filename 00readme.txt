shogi	ver1.0

将棋のゲームエンジンとGUIです．
GUIや棋譜形式の入力で駒を操作できます．

# 必要環境

* python 3系
* pygame

# モジュールのインストール

---------------------------
pip install pygame
---------------------------

# 使い方

* GUIで操作する場合
---------------------------
cd programs
python shogi_GUI
---------------------------

* 棋譜入力で操作する場合
未実装

# 注意点

将棋エンジンには一部未実装の機能があります．
・千日手の判定
・持将棋
・入玉時の宣言法
